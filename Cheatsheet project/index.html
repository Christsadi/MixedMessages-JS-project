HTML5 Cheatsheet

Standard Structure

Tag Name Description
<!DOCTYPE html> DOCTYPE HTML5 documents start with a <!DOCTYPE> declaration.
<html></html> html This tag is used at the begining and end of an HTMl document.
<head></head> head Use this tag to insert informations to the page.
<meta/> meta Includes "off-page" information and additional information about the page.
<base/> base Specifies the base URL and/or target for all realive URLs in a document.
<title></title> title The title tag is used to define the title of HTML document.
<link/> link Creates association with external pages like style sheets.
<script></script> script All the information or links to external scripts
<body></body> body defines the main content of an HTML document which dislpays on the browser.
